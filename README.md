# Trader Joe's Paradox

Portfolio analysis: Why does Trader Joe's win with fewer stores, fewer SKUs, and near-zero ad spend?

## Thesis
Fewer stores, fewer SKUs, near-zero ads → highest customer satisfaction + sales per square foot. Flaws function as moats.

## Phases
1. KPI Scorecard — quantify the paradox
2. Sentiment Teardown — Reddit + Yelp customer voice
3. SKU & Pricing Benchmark — private-label moat
4. Efficiency Model — margin waterfall
5. Expansion Gap — geographic whitespace

## Tools
Python (pandas, praw, vaderSentiment, scikit-learn, folium) · Tableau · Excel · SEC EDGAR · Reddit API · Census API
